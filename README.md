# CRM Система для Арбитражного Отдела

Фронтенд приложение для управления арбитражным отделом с системой ролей и детальной статистикой.

## Возможности

### Система ролей
- **Медиа баер** - доступ к собственной статистике
- **Тим-лид** - управление командой и статистика команды
- **Овнер/Админ** - полный доступ ко всем функциям и админ панель

### Функционал
- Регистрация и авторизация пользователей
- Управление командами (создание, редактирование, назначение участников)
- Управление Telegram каналами
- Детальная статистика по направлениям:
  - ФБ закуп
  - ТГ закуп
  - УБТ
  - Инфлюенс
  - Мотив
  - Реклама
- Статистика по заявкам, отпискам, подписчикам, тратам и доходам
- Управление рекламными слотами с детализацией по каналам и времени
- Админ панель с обзором всей системы

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Откройте браузер по адресу: http://localhost:3000

## Технологии

- React 18 с TypeScript
- Material-UI для интерфейса
- Recharts для графиков
- React Router для навигации
- Context API для управления состоянием

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── LoginForm.tsx    # Форма входа
│   ├── RegisterForm.tsx # Форма регистрации
│   ├── Dashboard.tsx    # Главный дашборд
│   ├── StatisticsOverview.tsx # Обзор статистики
│   ├── TeamManagement.tsx # Управление командами
│   ├── ChannelManagement.tsx # Управление каналами
│   └── AdminPanel.tsx   # Админ панель
├── contexts/            # React контексты
│   └── AuthContext.tsx  # Контекст аутентификации
├── types/               # TypeScript типы
│   └── index.ts         # Определения типов
├── data/                # Демонстрационные данные
│   └── demoData.ts      # Моковые данные
└── utils/               # Утилиты
```
